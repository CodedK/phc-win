# PHC-WIN #

## What is it? ##

PHC-WIN is a php complier/embeder. It takes [multiple] files and combines them into a exe, and embeds php into the exe allowing resdistributable PHP applications.

## Supported OS's ##
  - Windows XP
  - Windows Server 2003
  - Windows Vista
  - Windows 7
  - Windows 8
