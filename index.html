PHC-WIN

What is it?

PHC-WIN create single executables out of you PHP applications, and includes the WinBinder & win32std extensions.

Supported OS's

  • Windows XP (w/ self compiled PHP versions)
  • Windows Server 2003
  • Windows Vista
  • Windows 7
  • Windows 8
